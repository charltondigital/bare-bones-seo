A lightweight, performance-first SEO utility that provides absolute technical control over indexing assets without background bloat.

== Description ==

Bare Bones SEO is the ultimate antidote to massive, heavy alternative SEO suites. Built specifically for agencies, developers, and professional SEOs, this plugin drops the tracking scripts, heavy database junk, and real-time dashboard lag. 

Designed strictly for fresh website installs, it gives you direct control over indexing, sitemaps, titles, and descriptions across your entire site layout in a clean, lightning-fast workflow.

=== Key Features ===
* **Unified Page-Level Workspace:** A branded, compact meta box at the bottom of your editor containing title, description, custom JSON-LD schema, and indexing controls.
* **Live Visual Snippet Preview:** Simulates both desktop (600px) and mobile (360px) search appearances using system fonts, updating dynamically on your command.
* **Global Site Indexing Map:** A macro dashboard that strips away domain noise and cleanly organizes sitemap and indexing configurations by section.
* **Bulk Page Manager:** A fast, spreadsheet-style grid to audit and edit your entire site's indexing metadata in seconds using AJAX quick-saving without opening dozens of tabs.

== Installation ==

1. Upload the `bare-bones-seo` folder to the `/wp-content/plugins/` directory, or install directly through the WordPress admin panel.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Ensure no other major SEO suites are active; Bare Bones SEO is optimized strictly for fresh, clean environments.
4. Navigate to the new settings dashboard to configure your Global Site Indexing Map.
